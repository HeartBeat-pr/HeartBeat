# HeartBeat - Healthcare Management Platform

A web-based healthcare management platform focused on accessibility and ease of use.

## About

HeartBeat addresses the complexity and accessibiiity challenges of current healthcare platforms (such as the NHS website) by providing a clean, intuitive, and fully accessible interface focused on four core features.

## Four Pillars

**Book & Manage Appointments** 
**View Medical History**
**Order & Collect Prescriptions**
**Mental Health & Awareness**

## Tech Stack

**1. FrontEnd:** 
	HTML5, CSS3, JavaScript
**2. BackEnd:**
	Python
